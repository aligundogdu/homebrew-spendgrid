# Homebrew Tap for SpendGrid

Homebrew formula for [SpendGrid](https://github.com/aligundogdu/SpendGrid) - Financial Projection and Cash Flow Management CLI Tool.

## Installation

```bash
# Add the tap
brew tap aligundogdu/spendgrid

# Install SpendGrid
brew install spendgrid

# Upgrade to latest version
brew upgrade spendgrid
```

## Uninstallation

```bash
brew uninstall spendgrid
brew untap aligundogdu/spendgrid
```

## System Requirements

- macOS 10.15+ (Intel or Apple Silicon)
- Linux (x86_64)

## About SpendGrid

SpendGrid is a local-first, file-based financial management tool designed for people who want complete ownership of their financial data. All your data stays on your local machine in plain-text Markdown files.

Visit the [main repository](https://github.com/aligundogdu/SpendGrid) for documentation and usage examples.
